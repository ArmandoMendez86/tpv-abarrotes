Inicio de sesión

La app maneja dos roles: Administrador Vendedor

Captura usuario Ej. admin o cajero
Captura contraseña / PIN En algunos cajeros se usa PIN de 4 dígitos para entrar más rápido.
Entrar La app te mandará al módulo correspondiente según el rol.
Nota (offline): Si el dispositivo no tiene internet, la app sigue funcionando.
🧾

Vendedor (Caja)

Flujo recomendado para cobrar rápido y sin errores.

Buscar producto Por nombre, categoría o código de barras (si existe).
Agregar al carrito Ajusta cantidades desde el carrito si es necesario.
Cobrar Usa el botón principal (verde) para finalizar el ticket.
Consejo Si el cliente paga en efectivo, captura el monto recibido para calcular el cambio.
🧭

Administrador — Panel

El panel muestra accesos directos con iconos: productos, ventas del día, inventario, reportes, cajeros, ticket e impresora y control de caja.

Control de caja

  • Abrir caja: iniciar turno con fondo.
  • Movimientos: registrar entradas/salidas de efectivo.
  • Cerrar caja: arqueo y cierre del turno.
Importante Evita cerrar la app durante el cierre de caja.
🏷️

Catálogo — Crear / Editar producto

Campos principales:

  • Nombre: se recomienda en MAYÚSCULAS para búsquedas rápidas.
  • Código de barras: opcional; sirve para identificar rápido el producto.
  • Categoría: útil para filtros y reportes.
  • Costo / Precio: para margen estimado.
  • Stock / Stock mínimo: alertas de inventario.

Escanear código de barras

Toca “Escanear con cámara” Se abrirá el lector de códigos.
Apunta al código Mantén el teléfono firme y con buena iluminación.
Listo El número se pegará automáticamente en el campo “Código de barras”.
Si no escanea Limpia la cámara, mejora iluminación y evita reflejos. Si el código está dañado, captura el número manualmente.
📈

Reportes

  • Ventas del día: tickets y total vendido.
  • Top productos: productos con más movimiento.
  • Ventas por empleado: desempeño por cajero.
  • Tendencia: comparación mensual.
Tip Usa “Stock mínimo” para surtir antes de quedarte sin producto.
👥

Administración de cajeros

Alta rápida y control de acceso.

  • Crear vendedor: usuario + PIN (recomendado) o contraseña.
  • Dar de baja / Reactivar: el historial se conserva.
  • Cambiar PIN / Contraseña: sin perder ventas anteriores.
Recomendación Para operación rápida, usar PIN de 4 dígitos.
❓

Preguntas frecuentes

¿Mis datos se pierden si no hay internet?

No. La app funciona offline y guarda todo localmente.

¿Qué pasa si desinstalo la app?

Al desinstalar se borra la base local del dispositivo. Recomendación: no desinstalar sin respaldo.

¿Cómo actualizo la app sin perder datos?

Instala la actualización encima (mismo paquete y firma). No desinstales antes de instalar la nueva versión.
